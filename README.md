# SignalR 示例程序

> SignalR 用于创建实时Web应用程序。

![demo.gif](/images/demo.gif)

## 启动服务端

```
cd signalr-demo-server
dotnet build
dotnet run
```

## 启动客户端

```
cd signalr-demo-app
ng serve
```

## 启动浏览器

```
http://localhost:4200
```

参考链接：https://docs.microsoft.com/zh-cn/aspnet/signalr/overview/getting-started/tutorial-high-frequency-realtime-with-signalr
